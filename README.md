# Student Task Tracker

## Project Description
Student Task Tracker is a web-based task management system built with ASP.NET Core MVC and a separate ASP.NET Core Web API microservice. It supports authentication, role-based access, dashboards, task management, projects, quotes, and notes.

## Technologies Used
- ASP.NET Core MVC
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- ASP.NET Identity
- Bootstrap

## Architecture
User → MVC App → API Microservice → Database

## Features
- Register / Login
- Roles: Admin, User, Viewer
- Dashboard
- Task CRUD
- Search and Filter
- Viewer email search
- Projects
- Quotes
- Notes
